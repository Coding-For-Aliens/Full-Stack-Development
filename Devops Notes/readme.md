Breaking down things in simple words :

* Containerization - Docker
* Container Orchestration -K8/Kubernetes
* Database -SQL/DynamoDB
* Configration -Ansible/Terraform
* Monitoring - Promethus
* Logging -ELK/Splunk
* Bash scripting
* Programming Language - Python
